# Presupuesto de consultoría (PDF) con Python

Este proyecto genera un **PDF tipo “presupuesto de consultoría”** usando Python, colocando:
- Una **imagen de plantilla** (PNG) como fondo
- Texto encima (descripción, horas, valor/hora, tiempo, total)

## Archivos
- `proyecto.ipynb`: cuaderno con el código
- `william.png`: plantilla/fondo del diseño
- `Presupuesto.pdf`: resultado generado

## Requisitos
- Python 3.x
- Librería:
  - `fpdf`

Instalación:
```bash
pip install fpdf
